# Moon Reps - AI Agent Configuration

Este archivo configura las habilidades (skills) del agente AI para mejorar el diseño, rendimiento y aplicar buenas prácticas al proyecto Moon Reps.

## Stack Tecnológico

- **Framework**: Next.js 15
- **Estilos**: Tailwind CSS 3.4
- **Base de datos**: Supabase (PostgreSQL)
- **Lenguaje**: JavaScript (React)

## Skills Instalados

### Diseño y UI/UX
- `web-design-guidelines` - Directrices modernas de diseño web
- `frontend-design` - Mejores prácticas de diseño frontend
- `ui-ux-pro-max` - Guías avanzadas de UI/UX

### Framework y Código
- `next-best-practices` - Buenas prácticas específicas de Next.js
- `vercel-react-best-practices` - Mejores prácticas de React (Vercel)

### Base de Datos
- `supabase-postgres-best-practices` - Optimización de Supabase/PostgreSQL

### Auditoría y SEO
- `audit-website` - Auditoría completa del sitio web
- `seo-audit` - Mejoras de SEO y posicionamiento

## Áreas de Enfoque

1. **Rendimiento**: Optimización de carga, lazy loading, bundle size
2. **SEO**: Meta tags, estructura semántica, Core Web Vitals
3. **Diseño**: Consistencia visual, accesibilidad, responsive design
4. **Código**: Patrones de React, Server Components, best practices
5. **Base de datos**: Queries optimizados, indexación, caching
